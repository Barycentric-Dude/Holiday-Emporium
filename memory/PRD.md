# Holiday Emporium (Sachin Travels) - PRD

## Original Problem Statement
Re-imagine the WordPress-based Sachin Travels / Holiday Emporium site into a high-performance bespoke web app with premium travel aesthetic, bilingual support, and lead management.

## Architecture
- **Frontend**: React 19 + Tailwind CSS + Shadcn UI + React Router
- **Backend**: FastAPI + MongoDB (Motor async driver)
- **Design**: Organic & Earthy theme (Cormorant Garamond + Manrope)

## Pages
1. **Home** (`/`): Hero with mode toggle, tour categories bento grid, featured packages with itinerary accordions, Explore Asia carousel, trust indicators, company story, lead form, footer
2. **About** (`/about`): Hero, travel experiences grid, founder's story, value props, contact info, CTA banner, FAQ accordion

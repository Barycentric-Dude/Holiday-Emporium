# Holiday Emporium (Sachin Travels) - PRD

## Architecture
- **Frontend**: React 19 + Tailwind CSS + Shadcn UI + React Router
- **Backend**: FastAPI + MongoDB (Motor async driver)

## Pages
1. **Home** (`/`): Hero, tour categories, featured packages, explore asia, trust, story, lead form
2. **About** (`/about`): Hero, experiences grid, founder story, value props, contact, FAQ
3. **Ashtavinayak** (`/tours/ashtavinayak`): Hero, about journey, quickinfo, 8 temples grid + 2 extra stops, itinerary accordion, CTA

## Navigation
- Navbar: Tours dropdown with Domestic Tours > Ashtavinayak Yatra, International > Coming soon
- Mobile: Collapsible Tours section in hamburger menu
